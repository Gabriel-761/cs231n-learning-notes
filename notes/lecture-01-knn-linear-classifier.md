## 1. 核心知识点

### 1.1 图像分类与"语义鸿沟"

**一句话**：给定一张图，从预定义类别里选一个标签。

**展开**：机器眼里的图不是"一只猫"，而是一个张量，例如 `800 × 600 × 3`（宽 × 高 × RGB 三通道），每个像素值 0~255（8 bit，$2^8=256$）。人看到语义、机器看到数字，这道鸿沟叫 **Semantic Gap**。分类难，是因为视角、光照、遮挡、形变等变化下，**语义不变但像素剧烈变化**。

**类比**：你读文章看到的是"意思"，机器一开始只看到一堆编码数字，必须从数字里学出规律。

### 1.2 数据驱动方法

**一句话**：不手写规则，而是用大量"图像 + 标签"训练出一个分类函数。

**展开**：三步——收集数据集 → `train(images, labels) → model` → `predict(model, new_images) → labels`。好模型的标准是能**泛化**到没见过的图，而不是背下训练集。

### 1.3 最近邻 / kNN

**一句话**：测试图最像训练集里哪张，就用那张的标签；kNN 则看最近 k 张投票。

**展开**：
- 训练 ≈ O(1)（只是记住数据），但预测 O(ND)，很慢；需存全部数据 O(ND)。
- L1 距离：$d(I_1, I_2) = \sum_p |I_1^p - I_2^p|$。
- 引入 k 是为了抗噪声：k=1 对异常点极敏感，k 太大边界又太糊。

**类比**：不总结规律，只把所有例题背下来，考试时翻最像的旧题照抄——记忆成本高、查找慢、表面像就会答错。

### 1.4 线性分类器

**一句话**：用线性函数 $f = Wx + b$ 把像素映射成每个类别的分数。

**展开**：CIFAR-10 中 $x \in \mathbb{R}^{3072}$（32×32×3），$W \in \mathbb{R}^{10\times3072}$，$b \in \mathbb{R}^{10}$，输出 10 个类别分数（logits）。三种视角：代数（矩阵乘法）、视觉（每类一个模板）、几何（在空间里画超平面，b 负责平移边界）。**局限**：只能处理线性可分，搞不定 XOR、环形分布。

### 1.5 Softmax + 交叉熵损失

**一句话**：把 logits 变成概率分布，再用正确类别的负对数概率作为损失。

**展开**：
- Softmax：$P(y=k) = \dfrac{e^{s_k}}{\sum_j e^{s_j}}$（先指数化使其为正，再归一化使其和为 1）。
- 损失：$L_i = -\log P(y_i \mid x_i)$。答对越自信损失越小，答错越自信惩罚越大。
- 初始 sanity check：C 类随机初始化时 $L_i \approx \log C$，CIFAR-10 即 $\log 10 \approx 2.3$。

**类比**：损失函数像老师批卷子打分，优化算法像学生照着错处改 W、b。

## 2. 我的疑问与突破 ⭐

**疑问 1**：那句"初始 loss 偏离 2.3 可能是初始化/标签/Softmax/损失函数实现出问题"里，"实现出错"到底是结果错还是方式错？
**怎么想通的**："实现出错"指的是**代码逻辑/计算方式写错了**，结果错只是它的后果，二者不是并列关系。唯一的特例是"公式写对但数值不稳定"（如 logits 太大导致 $e^{s}$ 溢出成 NaN），这才是"方式对、结果错"——也正是要用"先减最大值再 Softmax"技巧的原因。

**疑问 2**：为什么要初始化？怎么从初始化走到最佳参数？
**怎么想通的**：损失对 W 没有闭式解，只能"先随机站一个起点，再一小步步往下坡走"。这就是**梯度下降**：前向算 loss → 算梯度（最陡下坡方向）→ 按 $W \leftarrow W - \eta\,\partial L/\partial W$ 更新，反复迭代直到收敛。用小随机数初始化是为了**打破对称性**，让各参数能分化。

**疑问 3**：学习率 η 怎么确定？
**怎么想通的**：没有公式，靠实验——看 loss 曲线（震荡/发散=太大，下降太慢=太小），按 10 的幂搜数量级（1e-1、1e-2、1e-3…）锁定后微调，进阶时再让它随训练衰减。它是个要在验证集上调的超参数。

## 3. 易混淆点辨析

| 概念 A | 概念 B | 关键区别 |
|--------|--------|----------|
| 参数 Parameter | 超参数 Hyperparameter | 前者模型自己学（W、b）；后者人提前设（k、η、正则强度） |
| 验证集 Validation | 测试集 Test | 验证集用来调参；测试集只在最后评估一次，绝不能用来调参 |
| L1 距离 | L2 距离 | L1 对坐标轴敏感（菱形边界）；L2 旋转不变、更平滑（圆形边界） |
| 非参数方法（kNN） | 参数化方法（线性） | 前者存全部数据、预测慢；后者只存参数、预测快 |
| 计算方式错 | 数值不稳定 | 前者是代码 bug；后者公式对但溢出/下溢（NaN、log0） |

## 4. 一页速记

- 图像 = Tensor；难点 = 像素变化大但语义不变（语义鸿沟）。
- kNN：训练快预测慢；L1 $=\sum_p|I_1^p-I_2^p|$；k 与距离函数都是超参数。
- 调参三件套：训练集学习 / 验证集调参 / 测试集最终考试。
- 线性分类器：$s = Wx + b$；CIFAR-10 中 $W \in \mathbb{R}^{10\times3072}$。
- Softmax：$P(y=k)=\frac{e^{s_k}}{\sum_j e^{s_j}}$；交叉熵 $L_i = -\log P(y_i)$。
- 初始 loss ≈ $\log C$；CIFAR-10 ≈ 2.3（sanity check）。

## 5. 输出检验：自测题

**【简单】** 两张 2×2 灰度图 A=[10,20;30,40]、B=[12,18;35,33]，求 L1 距离。
<details><summary>参考答案</summary>

|10−12|+|20−18|+|30−35|+|40−33| = 2+2+5+7 = **16**。

</details>

**【中等】** logits 为 cat=2.0, dog=1.0, frog=0.1，真实标签 cat，求 Softmax 概率与交叉熵损失。
<details><summary>参考答案</summary>

$e^{2.0}≈7.39,\ e^{1.0}≈2.72,\ e^{0.1}≈1.11$，分母≈11.22。
P(cat)≈0.66, P(dog)≈0.24, P(frog)≈0.10；$L_i=-\ln(0.66)≈0.42$。

</details>

**【困难】** 证明线性分类器无法分开 XOR：A 类 {(0,0),(1,1)}、B 类 {(0,1),(1,0)}。
<details><summary>参考答案</summary>

需同时满足 b>0、w₁+w₂+b>0（A 正）与 w₂+b<0、w₁+b<0（B 负）。由后两式得 w₁<−b、w₂<−b，相加得 w₁+w₂<−2b，故 w₁+w₂+b<−b<0，与"w₁+w₂+b>0"矛盾。加一个 x₁x₂ 特征后即可线性可分——这正是"线性层+非线性→神经网络"的动机。

</details>
